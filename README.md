### From VS Code Marketplace

1. Open VS Code
2. Go to **Extensions** (`Ctrl+Shift+X`)
3. Search for `Geode Amethyst`
4. Click **Install**
5. Go to **File** â†’ **Preferences** â†’ **Color Theme**
6. Select one of the Geode Amethyst themes

### Manual Installation

1. Download or clone the repository
2. Copy the `geode-amethyst-theme` folder to:
   - **Windows**: `%USERPROFILE%\.vscode\extensions\`
   - **macOS/Linux**: `~/.vscode/extensions/`
3. Restart VS Code
4. Select the theme in settings
